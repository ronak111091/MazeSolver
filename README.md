# MazeSolver
A simple java command line program that solves a maze.

**Note:
1st line of the input denotes the number of rows in the maze. <br />
S represents start of the maze.<br />
F represents end of the maze.
<br />

Example
<br />
Input:<br />
5<br />
@S@@@@<br />
@++@@@<br />
@@+@@@<br />
@++@@@<br />
@F@@@@<br />
<br />
Output:<br />
DRDDLD<br />
